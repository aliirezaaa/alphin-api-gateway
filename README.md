
# Alphin API Gateway

The API gateway 

## Features

- CRUD operations
- Error handling
- ready for production


## Run Locally

Clone the project

```bash
  git clone https://github.com/aliirezaaa/alphin-api-gateway.git
```

Go to the project directory

```bash
  cd alphin-api-gateway

```

Install dependencies

```bash
  npm install
```



Run the microservice

```bash
  nest start 
```
